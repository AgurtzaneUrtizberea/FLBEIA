## Installation

**TO INSTALL** this package, start R and enter:

	install.packages(repos="http://flr-project.org/Rdevel")

or, to install the development version

	library(devtools)
	install_github("FLBEIA", "flr")

**TO CITE** this package, start R and enter

	library(FLBEIA)
	citation("FLBEIA")
